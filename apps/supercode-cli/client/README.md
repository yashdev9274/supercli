# Supercode Terminal Client

Web-based terminal interface for Supercode AI coding agent.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Auth**: Better-Auth
- **State**: TanStack Query

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.
