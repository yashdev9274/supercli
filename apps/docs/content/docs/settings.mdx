# Settings

The Settings page allows you to manage your profile and connected repositories.

## Profile

Your profile information is pulled from GitHub:
- Avatar (from GitHub profile picture)
- Name
- Email
- Username

Profile information is read-only and updates automatically when you modify your GitHub profile.

## Connected Repositories

The Settings page also shows a list of all repositories you've connected to Supercode. From here you can:

- View all connected repositories
- See connection status
- Disconnect repositories you no longer need

## Account

### Signing Out

To sign out of your account:
1. Click your profile avatar in the header
2. Select **Sign Out**
3. You'll be redirected to the home page

### Deleting Account

To delete your account and all associated data:
1. Contact support through GitHub issues
2. We will process your request within 7 days

## Security

- All authentication is handled via GitHub OAuth
- No passwords are stored locally
- Session tokens are encrypted and secure
